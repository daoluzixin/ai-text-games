# 目元信息

## 项目名称

AI 文游游戏厅（ai-text-games）

## 项目简介

一个基于 Web 的 AI 文字冒险游戏平台。用户上传 `.docx` 格式的游戏剧本文件，系统将其内容作为 system prompt 注入 LLM，实现交互式文字冒险游戏体验。

## 当前阶段

MVP 已上线运行。核心功能（游戏加载、流式对话、会话持久化、文件上传）均已完成并部署到生产环境。

## 核心概念

| 概念 | 说明 |
|------|------|
| 游戏（Game） | 一个 `.docx` 文件，内容作为 LLM 的 system prompt |
| 会话（Session） | 一次游戏对话的完整历史，持久化到 SQLite |
| 游戏厅 | 前端首页，展示所有可用游戏和历史存档 |
| 头像（Avatar） | 每个游戏根据 game_id 哈希稳定分配一个 SVG 头像 |
